# 🏛️ Stoic Quotes - Daily Wisdom Dashboard

Start your day with profound philosophical insights! **Stoic Quotes** is a web application that fetches timeless stoic quotes and lets you curate your own vault of wisdom.

![Project Status](https://img.shields.io/badge/Status-Active-brightgreen) ![License](https://img.shields.io/badge/License-MIT-blue)

## ✨ Features

- **Consult the Oracle**: Fetch a fresh, random philosophical quote instantly from the [DummyJSON API](https://dummyjson.com/).
- **Vault Quotes**: "Vault" quotes 📍 to save them to your personal collection of carved wisdom.
- **Persistent Storage**: Your vaulted quotes are automatically saved to your browser's LocalStorage, so they're always there when you return.
- **Beautiful UI**: A clean, modern, dark-themed interface built with Tailwind CSS that evokes the feeling of ancient marble and wisdom.

## 🛠️ Tech Stack

- **Frontend Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **API**: [DummyJSON Quotes](https://dummyjson.com/docs/quotes)

## 🚀 Getting Started

Follow these simple steps to run the project locally:

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm run dev
   ```

3. **Open your browser**
   Navigate to the local URL shown in your terminal (usually `http://localhost:5173`).
